# Skill: Coherence Audit (PCD)

Esta habilidad materializa el **Protocolo de Coherencia Documental (PCD)**. Su propósito es actuar como un sistema inmunológico que detecta y reporta discrepancias entre las diversas "Fuentes de Verdad" del Terroir.

## Funciones:
1. **Auditoría de Roadmap:** Verifica que las fases marcadas como completadas en `ROADMAP.md` no tengan tareas pendientes críticas.
2. **Sincronía de Índices:** Asegura que los proyectos listados en `proyectos/gemini.md` existan físicamente en el disco.
3. **Validación de Versiones:** Detecta si hay múltiples versiones de un mismo protocolo o tratado causando confusión.

## Uso:
- El agente debe invocar esta skill antes de realizar un release o después de una refactorización masiva.
- Puede ser invocada por el usuario para obtener un "Reporte de Salud Ontológica".
